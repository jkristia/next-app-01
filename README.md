## Testing

Unit tests use [Vitest](https://vitest.dev/).

Run all tests:

```bash
npm run test
```

Watch mode:

```bash
npm run test:watch
```

Test files live alongside source under `app/**/*.test.ts` or `app/**/*.test.tsx`.
